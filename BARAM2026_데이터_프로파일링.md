## 1. 기초 도메인 지식

### 1.1 에너지 생성 원리
```
바람(공기의 운동에너지) → 날개가 받아 회전(기계에너지) → 발전기(전기에너지) → 전력망
```
- **로터(rotor)** = 날개(블레이드) + 중심축(허브). 실제로 회전하는 부분(회전체)

### 1.2 핵심 공식
```
P = (1/2) × ρ × A × v³ × Cp
```
| 기호 | 의미 | 단위 |
|---|---|---|
| P | 터빈이 만드는 전력 | W |
| ρ (로) | 공기밀도 | kg/㎥ |
| A | 로터가 쓸고 가는 면적 | ㎡ |
| v | 풍속 | m/s |
| Cp | 효율 계수 | 무차원 |

- `(1/2)ρAv³`은 바람이 가진 **전체 에너지**이다. 전부 뽑으면 로터 뒤 공기 속도가 0이 되어야 하는데, 그러면 공기가 빠져나가지 못해 뒤가 막히게 된다.
- 이 이론적 한계가 **베츠(Betz) 한계 = 16/27 ≈ 59.3%** (Cp가 넘을 수 없는 상한).
- 실제 상용 터빈은 공기역학·기계 손실까지 빼서 최적 조건에서 **Cp ≈ 40~50%**.
- Cp는 상수가 아니라 풍속에 따라 변하는 함수 → 이것이 파워커브의 곡선 모양을 만든다.
- **v³(세제곱)인 이유**: 입자당 에너지(v²) × 초당 통과 질량(∝v) = v³. 풍속 2배 → 에너지 8배.

### 1.3 실제 운전 로직 (파워커브)
| 지점 | 우리 실측 (V126) | 무슨 일이 |
|---|---|---|
| 컷인 (~3m/s) | 0~3m/s 출력 0.0 | 바람이 약하면 아예 안 돎. 풍속의 25%가 4.1m/s 미만 — 시간의 약 1/4은 거의 놀고 있음 |
| 상승 (3~13m/s) | 17.8 → 523 | 풍속³ 법칙으로 급상승. 예보 오차에 가장 민감 |
| 정격 (~13m/s) | 13~15m/s 531 플랫 | 발전기 한계. 날개 각도(피치)를 틀어 바람을 흘려보내며 출력 고정 |
| 고풍속 저감 (15~20m/s) | 478 | 돌풍·진동 보호로 출력 저감 |
| 컷아웃 (~22m/s) | 20m/s+ 319 급락 | 태풍급이면 날개 세우고 완전 정지 — "꼬리 절벽" |

- **S자 + 꼬리 절벽** 모양 → 풍속→발전량이 **비선형**, 예측 오차의 대가(파워커브 기울기)가 구간마다 다르다.
- 무릎 구간(10~13m/s)은 기울기 최대 → 오차 대가 최악 (정산금 손실이 여기 집중).

### 1.4 지형과 배치
- **풍향장미**: 서남서~서 계열 바람이 시간의 57%, **에너지 기여로는 80%** (자주 + 강하게 붊: 서풍 평균 9.0m/s vs 동풍 5.1m/s).
- **터빈 배치**: 17기가 남북 능선을 따라 일렬 — 주풍향(서)과 직각으로 세워 앞 터빈이 뒷 터빈 바람을 가리는 **후류(wake) 손실**을 최소화한 설계.
- 같은 단지 안에서도 터빈별 이용률 23~40% → 능선 위 몇백 미터 차이로 바람 자원이 약 1.7배 차이. 산악 풍력의 특징이자 **격자 평균보다 격자 최대(max) 피처가 강한 이유**.
- **허브고도 117m**: 지면 마찰로 지상 10m 바람보다 상공 바람이 강하고 안정적 → GFS 100m·850hPa 바람이 유용한 이유.

### 1.5 이용률 (Capacity Factor)
- 그룹 평균 이용률 26~33%. 출력 분포가 강한 우측 치우침(중앙값 ≪ 평균): **대부분 시간은 작게, 가끔 몰아서 많이 생산**.
- 이 비대칭 구조가 예측을 어렵게 하고, 평가 산식의 10% 컷(저발전 시간 채점 제외)과 맞물린다.

### 1.6 요약
> 바람의 에너지는 풍속³에 비례하고, 터빈은 컷인–정격–컷아웃 로직으로 그것을 S자 곡선으로 수확하며, 능선 지형과 풍향이 바람의 질을 결정한다.

## 2. 데이터 활용 방향 (피처 엔지니어링)

### 풍향 → 입력 피처
- **왜 필요한가**: 풍속이 완전히 같아도 방향에 따라 발전량이 다르다.
  실측 확인 — 같은 풍속 구간에서 **서풍이 동풍보다 약 1.3배 출력** (동풍은 능선 뒤에서 넘어오며 난류가 심함).
  즉 풍속만으로는 발전량이 유일하게 정해지지 않으므로 풍향까지 함께 넣어야 정확하다.
- **방법**: 예보 u/v에서 풍향 각도를 계산(atan2) → **sin/cos 두 값으로 변환**해 피처로 입력.
  (359°와 1°는 같은 방향인데 숫자로는 정반대 → 원 위 좌표로 바꿔야 연속성 보존)
- 베이스라인에 없던 피처. permutation importance 상위권으로 실제 점수에 기여.

### 풍속 → 입력 피처
- 원본에는 풍속 컬럼이 없고 **u(동서)·v(남북) 성분**만 있다.
- **피타고라스로 풍속을 직접 계산**: `ws = √(u² + v²)`
- 베이스라인은 u/v 격자평균 원값을 그대로 썼으나 두 문제가 있다:
  1. 모델이 피타고라스를 스스로 재발견해야 하는 비효율
  2. **벡터 상쇄** — 방향 다른 바람을 u/v 상태로 평균하면 상쇄되어 풍속 과소평가 (동풍 5 + 서풍 5의 u 평균 = 0)
- 우리 방식: **격자별로 풍속을 먼저 계산 → 그다음 평균/최대/표준편차 집계**로 상쇄 방지.
- 추가로 풍속³(공식의 v³), 공기밀도(ρ=기압/(287×기온)), 출력 프록시(ρ·ws³)까지 물리량 분해.

### 3-2. SCADA_VESTAS

터빈 내장 센서의 **10분 단위 실측 운전 기록** (기상예보가 아닌 실측값). 태백가덕산 1단계 VESTAS V126 12기.

**주최측 메타데이터**
- 모델명: VESTAS = V126 (로터 직경 126m)
- 설비용량: **3.6MW/기** (정격 출력 = 최대 출력), 그룹1·2 = 21.6MW (3.6×6기)
- 단위 변환: 3.6MW → 3,600kWh/시간 → **10분당 3,600/6 = 600kWh**

**변수 구조 (37열)**

| 변수군 | 열 수 | 의미 | 단위/범위 |
|---|---|---|---|
| kst_dtm | 1 | 계측 시각 (10분 간격, 누락·중복 0) | KST |
| wtg01~12_power_kw10m | 12 | 10분간 발전량 (호기별) | kWh, 정상 0~602 |
| wtg01~12_ws | 12 | 나셀 풍속계 실측 풍속 | m/s, 0~31.1 |
| wtg01~12_wd | 12 | 실측 풍향 | 도(°), 0~359 |

**출력 분포 — 강한 우측 치우침**
- 중앙값(95)이 평균(192)의 절반 — 대부분 시간은 조금, 가끔 몰아서 대량 발전.
- 0 이하 출력이 시간의 19~28% (무풍+정지 포함) → 비대칭 구조.

**파워커브 (풍속 구간별 평균 출력, kWh/10분)**

| 풍속 구간 | 평균 출력 | 해석 |
|---|---|---|
| 0~3 m/s | 0.0 | 컷인 미만 — 발전 불가 |
| 3~5 | 17.8 | 기동 구간 |
| 5~9 | 92→233 | 급상승 구간 (풍속³ 법칙) |
| 9~13 | 417→523 | 정격 접근 |
| 13~15 | 531 | 정격 포화 (플랫) |
| 15~20 | 478 | 고풍속 출력 저감 (돌풍 보호) |
| 20~35 | 319 | 컷아웃 근접 — 안전 정지 섞임 |

- 15m/s 이상에서 출력 감소 = 태풍급 바람에서 터빈이 스스로 출력을 줄이거나 정지.

**풍향**: 서남서~서 계열이 지배적(에너지의 80%), 북동풍은 부차적(겨울 계절풍).

**결측치 & 이상치**

**1. 수천만 부호쌍 오류 868개 — 전처리함**
- 정격 3,600kW인데 **1,363만** 같은 값(±5,177만까지) — 물리적으로 불가능, 음수도 존재.
- 음수·양수가 거의 같은 크기로 짝지어 나타남 = 전형적 **누적 계량기 오류** 패턴.
  ```
  10분 발전량 = 누적값(현재) - 누적값(10분 전)
  누적값:  13,634,466 → [오류:590] → [복구:13,634,465]
  차이:        -13,633,876(폭락)      +13,633,875(복구)
  ```
- 오염 효과: 정제 전 SCADA 합산 vs 라벨 상관 **0.001(무상관)** → 제거 후 **0.9998**.
- 잡은 방법: ① 터빈별 min/max에서 ±4,200만 발견 → ② 분위수 99.9%가 602 이하(극단값은 별개 집단) → ③ ±쌍 패턴 확인 → ④ **|값|>660(이론 최대의 110%)을 NaN 처리** (제거보다 NaN으로 두어 나중 활용 여지 보존).

**2. 풍속 센서 고착 의심 — 무시 가능**
- 0 아닌 동일값 최장 60분(wtg04, wtg06 각 1회). 몇 시간짜리 고착은 없음. 통계 영향 미미 → 미처리.

**3. 무풍인데 고출력(>500kW) — 무시 가능**
- 터빈당 0~5회 순간 글리치. 풍속계 오류 또는 타임스탬프 어긋남 추정. 규모 극소 → 미처리.

**결측치 & 이상치**

1. **결측 존재** — vestas와 달리 실제 결측 있음(power 최대 1.3%, ws 최대 1.4%). 시간 합산 시 자동 제외 → 별도 미처리.
2. **정격 초과값 482개** — 800/900/1000 라운드 값(양자화). 부호쌍 오류가 아니므로 NaN이 아닌 **735(정격 105%)로 클리핑**.
3. **풍향 규약 불일치** — -180~180° → (`% 360`)으로 0~360 통일 (vestas와 일치).
4. **라벨 정합성 — vestas보다 나쁨 (중요)**: SCADA 합산 vs kpx_group_3 상관 0.9966이나 **잔차 1,000kWh 초과가 6.1% 시간대** (vestas 그룹1·2는 0.0%). 결측 + 라벨 보정 흔적 → 그룹3 라벨 품질이 낮다는 증거. 이 6.1%를 학습에서 제외한 **라벨 정화**가 실험 최대 기여(+0.0031)였음.

### 3-4. LDAPS (국지 기상예보)

기상청 국지예보모델(LDAPS), 약 1.5km 해상도 **16개 격자**. 매일 09시 초기화, 13시부터 사용 가능, 다음날 24시간(01시~익일 00시) 예보만 추출. forecast_kst_dtm당 grid_id 16행.

**변수 구조 (35열 = 메타 5 + 기상 30)**

| 변수군 | 열 수 | 주요 변수 | 비고 |
|---|---|---|---|
| 메타/좌표 | 5 | forecast_kst_dtm, data_available_kst_dtm, grid_id, latitude, longitude | 예보/발표 시각·격자 |
| **바람** | 8 | 10m u/v, 50m U/V max·min, 5m 경계층풍(BLWS) | 풍속·풍향의 원천 |
| 기온/습도 | 4 | 2m 기온(t), 이슬점(dpt), 상대습도(r), 비습(q) | 공기밀도 계산 |
| 기압 | 2 | 지표기압(sp), 해면기압(prmsl) | |
| 복사/운량 | 7 | 단파복사 직달/산란, 순복사, 상·중·하층운량 | 풍력 무관, 대부분 미사용 |
| 강수/눈 | 5 | 강수율, 강설률, 강수량, 적설, 융설 | 풍력 무관 |
| 기타 | 4 | 경계층높이(blh), 육지마스크(lsm), 지표고도(h) | lsm은 상수 → 제거 |

- 핵심은 **바람 계열 8개** (허브고도 117m에 가까운 50m 바람이 최상).

**규모 및 결측**

| 구분 | train | test |

| shape | 420,864 × 35 | 140,160 × 35 |

2. **상대습도 > 100%** — train 11,654개(최대 113%), 과포화 출력(정상) → 100 클리핑.
3. **surface_0_lsm 상수(=1)** — 정보량 0 → 제거.
4. 50m 바람 max<min 모순 2~3개 — 무시 가능.

### 3-5. GFS (광역 기상예보)

미국 글로벌예보모델(GFS), 약 25km 해상도 **9개 격자**. forecast_kst_dtm당 grid_id 9행. **단지 인접 격자는 5번 하나뿐**(4.9km), 나머지 8개는 19~40km 밖.

**변수 구조 (40열 = 메타 5 + 기상 35)**

| 변수군 | 열 수 | 주요 변수 | 비고 |
|---|---|---|---|
| 메타/좌표 | 5 | forecast_kst_dtm, data_available_kst_dtm, grid_id, lat, lon | |
| **바람 (다중 고도)** | 8 | 10m, 80m, 100m u/v, 경계층(PBL) u/v | 100m가 허브고도 117m에 최근접 |
| 상층 바람/기온 | 11 | 850·700·500hPa u/v/기온, 500hPa 지위고도 | **850hPa 바람이 강신호 (상관 0.76)** |
| 기온/습도 | 4 | 2m 기온·이슬점·상대습도·비습 | |
| 기압 | 2 | 지표기압, 해면기압 | |
| 복사/운량/기타 | 6 | 하향단파(dswrf)·장파(dlwrf), 돌풍(gust), 운량 | 돌풍 유용 |
| 강수/환기 | 4 | 강수율, 총강수, 환기율(VRATE) | |

**규모 및 결측**

| 구분 | train | test |
|---|---|---|
| shape | 236,736 × 40 | 78,840 × 40 |
| 시각당 격자 | 9 일정 | 9 일정 |
| NaN | 0 | 0 |

**결측치 & 이상치 (모두 정상 판정)**
1. **VRATE(환기율) 최대 약 9만** — 원래 단위가 큰 정상 변수(중앙값 ~1,000), 이상치 아님.
2. **야간 dswrf > 0** — GFS 복사 변수가 직전 수 시간 구간 평균값이라 일몰 직후에 낮 시간이 포함된 것(정상). "순간값 아님"만 유의.
3. 결측·중복·예보 복붙 없음 — train/test 모두 완전.

## 4. test 데이터 (2025년, 예측 대상) 분포 진단

> 핵심 질문: **2025년 기상이 학습 기간(2022~2024)과 얼마나 닮았는가** = 일반화 가능성

### train vs test 풍속 분포 — 큰 도메인 시프트 없음

| 변수 | train 평균 | test 평균 | train q95 | test q95 |
|---|---|---|---|---|
| LDAPS 10m 풍속 | 4.84 | 5.11 | 9.89 | 9.94 |
| LDAPS 50m 돌풍 | 7.50 | 8.03 | 15.96 | 15.97 |
| GFS 100m 풍속 | 3.83 | 4.12 | 9.78 | 10.47 |
| GFS 850hPa 풍속 | 8.04 | 8.75 | 19.79 | 20.30 |

- **2025년이 학습 기간보다 바람이 약 5~9% 강함**, 단 상위 분포(q95)는 거의 동일 → 극단값 범위는 같고 평균만 소폭 상승.
- 함의: 모델이 이미 test의 강풍을 반영하므로, **예측을 인위적으로 위로 미는 상향 조작은 과예측이 되어 손해** (실제 실험에서 반복 확인됨).

### 무결성
- 격자 좌표·개수·시각이 train과 완전 동일. ldaps_test만 3시각 결측(보간), gfs_test는 완전무결.
